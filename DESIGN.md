# DESIGN.md

## 气质与意象
地下光纤管道中透出的冷光——干净、专业、值得信赖。像是一个现代化的网络运维控制台，而非花哨的消费级产品。界面克制但信息密度高，每一个像素都有目的。

## 视觉策略
- 深色主题为默认，模拟终端/运维场景的心理暗示
- 细微的网格纹理背景，暗示网络拓扑结构
- 卡片使用半透明毛玻璃质感，区分层级但不生硬

## 配色方案
- 背景：#0a0f1a（深海蓝黑，不是纯黑）
- 卡片：rgba(15, 23, 42, 0.8) 毛玻璃
- 主色/强调色：#06b6d4（Cyan-500，光纤冷光感）
- 次要强调：#22d3ee（Cyan-400，悬停态）
- 文字主色：#e2e8f0（Slate-200）
- 文字次要：#94a3b8（Slate-400）
- 边框：rgba(6, 182, 212, 0.2)（Cyan 低透明度）
- 成功态：#10b981（Emerald-500）
- 错误态：#ef4444（Red-500）

## 字体排版
- UI 文字：系统默认 sans-serif（Tailwind 默认）
- 代码/节点信息/编码结果：JetBrains Mono 或等宽字体
- 标题加粗醒目，表单标签紧凑

## 动效与交互
- 表单提交时按钮加载态旋转
- 二维码生成时淡入动画
- 复制按钮点击后短暂变绿反馈
- 卡片悬停微妙上浮 + 边框高亮

## 页面结构
- 顶部：标题 + 简短说明
- 主体左侧/上方：节点信息输入表单（协议选择 + 各字段）
- 主体右侧/下方：结果展示区（拼接预览 → Base64 编码结果 → 静态文件路径 → 二维码）
- 历史记录区（可选，本次暂不实现）

## 设计禁忌
- 不要使用蓝紫色渐变
- 不要使用纯白背景
- 不要过度装饰，保持工具感
- 节点信息展示不要用等比缩放，必须可复制可读
